FPS HUD Changelog

[v1.0.0] – 2025-12-30

Initial release

Displays real-time FPS on screen

Customizable position and scale of the HUD

Toggleable display with a default keybind (K)

Minimalistic, lightweight design for performance

[v1.0.1] – 2025-12-31

Added toggle button in the settings menu for easier on/off control

Improved default positioning for different screen resolutions

Performance optimizations for lower-end systems

Minor bug fixes

[v1.1.0] – 2026-01-05 (planned)

Added customizable font and color options

Added option to show average FPS over time

Smooth fade-in/out animation when toggling HUD
